# IOT_Blind_Stick
In order to integrate the hardwares for the system at first all of the required hardwares should be collected. All of the hardwares required for this system are as;
1.	Arduino Uno
2.	UltraSonic Sensor
3.	Buzzer
4. 9 V Battery
5. Battery Adapter
6. Jumper Wires
